FireTray

  • About
  • Release Notes

FireTray is a Linux Mozilla addon that targets Firefox, Thunderbird and Seamonkey. It provides a tray icon with much window handling functionality. For mail applications, the icon can show the number of unread messages. FireTray is very customizable.
